# Discord-Alfred
Un bot discord en Javascript

## Installation

Installez les dépendances

```
npm install
```

Démarrez le serveur

```
node bot.js
```

Pour toute information a propos des concepts et de la procédure d'installation et de configuration complète reportez vous à la [documentation](/doc).
